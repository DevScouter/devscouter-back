# Dev Scouter Serverside

Gives generic insight to a developer via Github profile

## Virtual Environment

For MacOS,

```bash
python3 -m venv .venv
source .venv/bin/activate
```

For Windows,

```bash
python3 -m venv .venv
.venv/Scripts/activate
```

## Install Dependencies

```bash
pip install -r requirements.txt
```
